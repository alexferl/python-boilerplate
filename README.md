# python-boilerplate
A minimalistic Python boilerplate with [black](https://github.com/psf/black) and [pre-commit](https://pre-commit.com/).

## Requirements
- Python 3.11+

## Using

```
$ git clone https://github.com/alexferl/python-boilerplate.git myapp
$ cd myapp
$ make dev
$ pipenv shell
```

To run the tests:

```
$ make test
```
